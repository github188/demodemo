VIP 功能设置
作家分成比:
电子书作品被购买时，作家所分成的反馈奖励。
当每被购买 1 {C_moneytitle} 时作家可得分成多少{C_moneytitle} 。默认为0.7。
VIP书籍过期时限(单位天):
超过此天数的vip章节即被自动转换为公开书籍。
VIP章节价格(单位网站虚拟币):
设定每千字的价格。用来自动对VIP章节定价。
VIP阅读版权 - 文字内容:
显示在vip章节的图片上。“[rd8vip]”将被订阅用户名自动替换。
VIP阅读版权 - 字体:
显示版权的字体大小。ttf文件，位于“data/font”目录下。
VIP阅读版权 - 字号:
显示版权的字体大小。
VIP阅读 - 每行字数:
VIP图片阅读时，每行显示的字数。
VIP阅读 - 字体:
VIP图片阅读时，显示的字体。ttf文件，位于“data/font”目录下。
VIP阅读 - 字号:
VIP图片阅读时，显示的字体大小。
VIP阅读 - 背景颜色:
VIP图片阅读时，图片背景色。
VIP阅读 - 背景颜色透明度:
0 为完全不透明，127 为完全透明。
VIP阅读 - 文字颜色:
VIP图片阅读时，显示文章的颜色。
VIP防盗水印 - 字体:
VIP图片防盗水印显示的字体大小。ttf文件，位于“data/font”目录下。
VIP防盗水印 - 字号:
VIP图片防盗水印显示的字体大小。
VIP防盗水印 - 颜色:
VIP图片防盗水印的颜色。建议尽量与背景色相近，但务必不要相同。
VIP章节提示购买 - 提示图片:
当用户尚未购买此章节时显示的提示图片，位于“images”目录下。
VIP章节提示购买 - 章节购买按钮坐标:
在上述提示图片中的提示章节购买按钮的坐标。
格式为四个数字坐标,半角逗号隔开,依次为左上x&y,右下x&y。
VIP章节提示购买 - 用户登录按钮坐标:
在上述提示图片中的用户登录按钮的坐标。格式同上。
VIP章节提示购买 - 在线支付按钮坐标:
在上述提示图片中的在线支付按钮的坐标。格式同上。
