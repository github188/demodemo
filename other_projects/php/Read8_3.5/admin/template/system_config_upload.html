图片上传设置
图片上传:
是否允许作家在发表作品的时候上传图片。
 
style="display: none;">
 图片大小(单位: Byte(s)):
 设置允许上传图片的最大限制，1024 Bytes 为 1 KBtyes，当前服务器上传最大限制为{upload_max_filesize}。
style="display: none;">
 图片类型:
 设置允许上传图片的格式，可多选。




style="display: none;">
 水印:
 对上传的图片附件自动加水印，并可设置水印的位置。

   
   
   
style="display: none;">
 水印方式:
 使用文字或图片作为水印。
 
style="display: none;">
 水印文字:
 该文字将附加在上传的图片上。
style="display: none;">
 水印文字颜色:
 水印文字的颜色。
style="display: none;">
 水印图片:
 该图片将附加在上传的图片上。
style="display: none;">
 水印图片透明度:
 水印图片的透明度。
