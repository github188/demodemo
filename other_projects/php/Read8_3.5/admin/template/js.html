常见出错信息
  • Bad Request. Lack of required parameters.              参数错误
  • Bad Request. Illegal request.                          校验码错误
  • Bad Request. Please choose a method.                   书籍类型与书籍分类均未定义
  • Bad Request. Please limit a number.                    调用数量未定义
  • Can't write cache file. Please check your permission.  写入缓存失败
  • Can't read cache file. Please check your permission.   读取缓存失败

书籍类型:
书籍类型与书籍分类至少设定一种。
书籍分类:
书籍类型与书籍分类至少设定一种。
调用数量:
设置调用多少条数据，必填。
页面编码:
如调用的页面不是utf-8编码且出现乱码则需设置此项，通常为gbk。
缓存更新时间(单位秒):
如未在常规选项中设置统一缓存时间请务必设置该项，否则将非常消耗服务器资源。
如已在常规选项中设置统一缓存时间则该项设置无效。
在新窗口打开页面:
是否在新窗口打开链接，默认为是
 
打开新窗口方式:
如果您明确知道您的页面符合 XHTML 1.1 或 XHTML 1.0 Strict 标准请选择标准方式，否则请使用默认兼容模式
