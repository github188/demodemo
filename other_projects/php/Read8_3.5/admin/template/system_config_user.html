用户基本设置
注册:
选中“关闭”选项将禁止游客注册成为网站会员，但不影响已注册会员继续使用网站相关功能。
 
style="display: none;">
注册审核:
选中“开启”选项后，新注册用户需要经过管理员的审核方可成为正式注册会员。
 
style="display: none;">
用户名最小长度:
注册用户名的最小长度，一个汉字长度为2，默认值为4。
style="display: none;">
用户名最大长度:
通常此值不应超过所有整合的系统(如:Discuz!、PHPWind等)允许用户名的最长值，例如整合的系统允许用户名长度最大值分别为15、20，则此值不应超过15，否则将可能导致部分通行证系统无法正常注册。当前可设置的最大值为200，默认值为12。
普通用户书架本数:
普通用户书架的书籍收藏量上限,即最多允许收藏的书籍本数。
