站点基本设置
网站名称:
网站名称，将显示在导航条和标题中。
网站地址:
部分跳转页面和导航条、静态阅读页面中显示。
地址需要包含“http://”，结尾不要带“/”。
论坛地址:
显示于用户导航菜单中。
相对或绝对路径均可。
网站版权信息:
支持 HTML 代码。其中，{SITE_NAME}将被替换成网站名称，{SITE_ADDR}将被替换成网站地址，{BBS_URL}将被替换成论坛地址，{NOW_YEAR}将被替换成当前年份。若要显示版权符“©” 请使用 “&copy;”标记。
程序版权信息:
在此输入程序的版权信息,支持 HTML 代码，如下所示：
例1: 填写“本站由{read8cn}提供技术支持”将显示“本站由读吧书网提供技术支持”
例2: 填写“Powered by {read8en} {read8ver}”将显示“Powered by Read8.Net 3.5 Beta 1”
去除程序版权信息授权码:
读吧书网版权所有，未经允许严禁删除官方版权信息，违者我们保留诉诸法律的权利。
如果您购买了官方授权去除读吧书网版权信息的服务，请点击这里到官方互动获得相应的授权码。
网站备案:
填入备案号显示于页脚，留空即不显示。
默认时区:
网站所使用的时区，与服务器时区无关，北京时间为GMT+8，当前时区: GMT{time_zone}。
时间校对(单位秒):
服务器时间不准确时使用它来校对时间，当前网站时间: GMT{time_zone} {TIMESTAMP_DATE_Y-m-d H:i:s}。
网站开启:
选中“关闭”选项，可将网站暂时将关闭，其他人无法访问，站长不受限制。
 
网站关闭原因:
网站关闭时的显示信息，支持 HTML 代码。
