• 这里设置的是首页一个模块的查询SQL语句及标识等信息。
    自定义缓存
    缓存标识:
    只能使用字母和数字。
    {id}
    缓存描述:
    说明该缓存有什么用处。
    SQL 语句:
    取得缓存结果的 SQL 查询语句。
    可以使用“{db_prefix}”代替数据表前缀。
    以下内容将被自动替换成新字段：
      id(替换成bid)
    以下内容将被自动替换成相应的中文:
      subject(被替换成该一级分类的名字)、
      content(替换成该二级分类的名字)、
      style(被替换成作品的风格)、
      state(作品的状态)、
      licence(作品的授权)
    如需要原值则需将该值在选取时映射别名。
    结果维度:
    生成一维或二维数组。
    style="display: none;">
    二维归类:
    二维数组中分类依据。
    style="display: none;">
    一维包含:
    半角逗号分隔。
