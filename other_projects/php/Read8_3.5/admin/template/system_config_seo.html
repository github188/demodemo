搜索引擎优化
书籍数据目录:
存放书籍静态文件的目录名，默认为“html”，若修改此处，请使用ftp或在服务器上将该文件夹同步改名。
书籍数据子目录:
存放书籍静态文件的子目录格式，默认为“首字母/日期”。修改后将应用于新书，原有书籍不受影响。
书籍数据子目录日期格式:
当上一项设置中存在日期时，该项有效。
标题附加字:
网页标题通常是搜索引擎关注的重点，本附加字设置将出现在标题中论坛名称的后面，如果有多个关键字，建议用 “|”、“,” 等符号分隔。

搜索关键字:
本项作用于网站动态页面的 meta keywords 标签中，正确的设置本项有利于用户从搜索引擎搜索到本站，多个关键字间请用半角逗号 "," 隔开。

网站描述:
本项作用于网站动态页面的 meta description 标签中，请用自然语言描述本网站，正确的设置本项有利于用户从搜索引擎搜索到本站，若留空则不在网站中使用 meta description，静态页面不受影响。
小说重写模式:
此项设置小说的重写模式。设置方式请务必参考读吧书网3.5重写规则。



作者重写模式:
此项设置作者的重写模式。设置方式同上。
